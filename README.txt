/////// READ-ME ////////////////////////////////////

Para utilizar o programa, inicialmente abra o arquivo de entrada "input.txt", e modifique a entrada colocando as posicoes dos rovers. Observacao: nao serao aceitas posicoes invalidas (fora do plateau), nem posicoes iguais, pois dois rovers nao podem ocupar a mesma posicao.

Compile os arquivos .java da seguinte maneira:
javac *.java

Execute o programa no seguinte formato:
java MarsRovers <nomeDoArquivodeInput>

Neste caso, o arquivo de input e chamado "input.txt", entao execute da seguinte maneira:
java MarsRovers input.txt

As posicoes finais de todos os rovers sao escritas no arquivo output.txt.
